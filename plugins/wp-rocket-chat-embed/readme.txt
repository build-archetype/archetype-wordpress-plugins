Easily embed Rocket.Chat in your WordPress site with simple shortcode integration.

== Description ==

Rocket.Chat Embed allows you to quickly integrate Rocket.Chat into your WordPress website using a simple shortcode. Perfect for adding chat functionality to any page or post.

**Free Features:**
* Basic Rocket.Chat embedding via shortcode
* WordPress user integration
* Admin dashboard configuration
* Chat state control (enable/disable)

**Premium Features:**
* Single Sign-On (SSO) with WordPress users
* Auto-login functionality
* Custom CSS styling
* Guest access support
* Priority support

**Usage:**
Use the shortcode `[rocketchat_iframe]` anywhere on your site to display the chat widget.

Optional parameters:
* `channel="general"` - Specify channel
* `width="100%"` - Set width
* `height="600px"` - Set height

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/rocket-chat-embed/`
2. Activate the plugin
3. Go to Settings â†’ Rocket Chat Embed

5. Use `[rocketchat_iframe]` shortcode in posts/pages

== Frequently Asked Questions ==

= Do I need a Rocket.Chat server? =
Yes, you need access to a Rocket.Chat server instance with admin credentials.

= Can I customize the appearance? =
Premium version includes custom CSS options for complete styling control.

= Does it work with any theme? =
Yes, the plugin is designed to work with any WordPress theme.
